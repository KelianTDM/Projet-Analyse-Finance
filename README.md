# Dashboard d'Analyse Financière - [Nom de votre Projet]

## Description du projet
Ce projet consiste en la création d'un tableau de bord décisionnel (Business Intelligence) pour le suivi de la performance financière. L'objectif est d'analyser le chiffre d'affaires, la croissance annuelle (YoY) et la contribution par catégorie de produits, afin de faciliter la prise de décision stratégique.

## Fonctionnalités
* **Extraction et Nettoyage :** Traitement des données brutes via SQL et Power Query.
* **Modélisation :** Architecture en étoile (Star Schema) avec une table de faits et une dimension temps dédiée.
* **Indicateurs de Performance (KPIs) :**
  * Chiffre d'Affaires Total (Gross Revenue).
  * Cumul Annuel (YTD - Year-To-Date).
  * Analyse de croissance Year-over-Year (YoY %).
* **Visualisation :** Cartes de performance, matrices annuelles et graphiques en cascade pour l'analyse contributive.

## Stack Technique
* **Base de données :** SQLite (DB Browser for SQLite)
* **Nettoyage :** SQL & Power Query
* **Modélisation & Dashboards :** Power BI Desktop
* **Langages :** DAX (Data Analysis Expressions)

## Analyse et Interprétation
Ce dashboard permet de répondre aux problématiques suivantes :
1. **Pilotage :** Visualiser rapidement les écarts de performance par rapport à l'année précédente.
2. **Performance :** Identifier les catégories de produits les plus rentables via le graphique en cascade.
3. **Réconciliation :** Garantir l'intégrité des données financières grâce à la modélisation en étoile.
